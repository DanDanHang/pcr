# 介绍

2017年最新全国省市区数据库。包含名称、城市代码、上级代码、级别、邮编。 

在应用程序中，省市区级联选择是比较常用的功能，这个功能在多数情况下，需要集成到自己的系统中，而不是调用一些地图提供方的API。 我创建这个库有以下目的：

1. 一个中心的位置，对数据提供持续更新。
2. 让开发能够便捷的将地址选择功能集成到自己的系统中。

# 贡献
